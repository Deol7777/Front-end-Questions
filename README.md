# Front End Questions

Practice repo for front-end coding questions and solutions.

## Structure

- `Javascript/` — JavaScript problems and solutions.

## Workflow

1. Paste a question/problem with your code.
2. A test function is added to the **same file** to verify the solution.
3. Your original code stays untouched (see [INSTRUCTIONS.md](INSTRUCTIONS.md)).

## Running

```bash
node Javascript/<file>.js
```
